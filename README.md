# 💙 Blue Lui - Personal Website

Personal portfolio website for **Blue Lui (LUI Ying Lam)**, a PhD Candidate in Cell and Molecular Biology at CUHK.

## 🌐 Live Website

- **Primary URL**: https://bluelui.pages.dev/ (Cloudflare Pages)
- **GitHub Repo**: https://github.com/lbmkyi1/blue-lui-website

## ✨ Features

- **Bilingual** (English & Traditional Chinese) with language toggle
- **Responsive Design** - mobile-first, works on all devices
- **7 Sections**:
  1. Hero (Home)
  2. About Me (關於我)
  3. Research & Skills (研究與技能)
  4. Publications (學術發表)
  5. Awards & Teaching (榮譽與教學)
  6. Beyond the Lab (實驗室之外)
  7. Contact (聯絡我)
- **Scroll Animations** - smooth fade-in effects
- **Sticky Navigation** with active section highlighting
- **Contact Form** with email integration
- **CV Download** button

## 🎨 Design System

| Element | Value |

## 🚀 Deployment

This is a static website that can be deployed to:
- **GitHub Pages**
- **Cloudflare Pages**
- **Netlify**
- **Vercel**

## 📝 Customization

### Adding Your Photo
Replace the placeholder in the Hero section:
```html
<div class="hero-image-placeholder">
    <!-- Replace this div with an <img> tag -->
    <img src="your-photo.jpg" alt="Blue Lui">
</div>
```

### Updating Publications
Edit the Publications section in `index.html` to add/remove papers.

### Updating CV
Replace `Blue_Lui_CV.docx` with your latest CV file.
